# Тестовое задание

React-приложение для выбора тарифов фитнес-сервиса с таймером, соглашением и функцией покупки.

## 🚀 Технологии

- **React 19** с React Compiler
- **TypeScript** для типизации
- **Vite** как сборщик
- **Redux Toolkit** для управления состоянием
- **SCSS** с модулями для стилей
- **SVG Sprites** для иконок
- **React Hooks** (custom hooks для тарифов и таймера)

## 📁 Структура проекта

```
src/
├── app/                 # Основное приложение и хранилище Redux
│   ├── App.tsx
│   └── store/
├── widgets/            # Виджеты (компоненты верхнего уровня)
│   ├── header/         # Шапка
│   └── tariffs/        # Блок тарифов
├── features/           # Бизнес-логика (хуки)
│   ├── tariffs/        # Логика работы с тарифами
│   └── timer/          # Таймер обратного отсчёта
├── shared/             # Общие ресурсы
│   ├── assets/         # Шрифты, изображения, SVG
│   ├── lib/            # Утилиты и хуки
│   ├── styles/         # Глобальные стили, переменные, миксины
│   └── ui/             # Переиспользуемые UI компоненты
└── main.tsx           # Точка входа
```

## 🎯 Функциональность

- **Отображение тарифов**: Карточки с деталями (цена, срок, скидки)
- **Таймер**: Обратный отсчёт для акционных предложений
- **Соглашение**: Чекбокс принятия условий перед покупкой
- **Валидация**: Блокировка кнопки покупки без выбора тарифа и согласия
- **Состояние загрузки/ошибки**: Индикаторы при получении данных
- **Адаптивный дизайн**: Поддержка мобильных устройств

## 🛠 Установка и запуск

### Предварительные требования
- Node.js (версия 18+)
- npm или yarn

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```
Приложение будет доступно по адресу [http://localhost:5173].

### Сборка для production
```bash
npm run build
```

### Превью собранного приложения
```bash
npm run preview
```

### Линтинг
```bash
npm run lint
```

## 🧩 Ключевые компоненты

### `Tariffs` (`src/widgets/tariffs/Tariffs.tsx`)
Основной компонент выбора тарифов. Содержит:
- Сетку карточек тарифов
- Таймер обратного отсчёта
- Чекбокс соглашения
- Кнопку покупки с валидацией

### `TariffCard` (`src/widgets/tariffs/TariffCard.tsx`)
Карточка отдельного тарифа с:
- Названием, ценой, сроком
- Индикатором выгоды
- Кнопкой выбора
- Стилями для выбранного состояния

### `useTariffs` (`src/features/tariffs/useTariffs.ts`)
Кастомный хук для управления тарифами:
- Загрузка данных
- Сортировка по выгоде
- Выбор тарифа
- Логика покупки

### `useTimer` (`src/features/timer/useTimer.ts`)
Хук таймера обратного отсчёта:
- Управление временем
- Форматирование в MM:SS
- Событие окончания таймера

## 🎨 Стилизация

- **SCSS модули**: Компонентные стили в `*.module.scss`
- **Глобальные переменные**: Цвета, шрифты в `src/shared/styles/_variables.scss`
- **Шрифты**: Gilroy, Montserrat, Raleway (подключены через `@font-face`)
- **Адаптивность**: Медиа-запросы и контейнеры

## 🔧 Конфигурация

### Vite (`vite.config.ts`)
- Алиас `@` для `src/`
- Плагин React Compiler
- Плагин SVG спрайтов

### TypeScript (`tsconfig.json`)
- Строгая типизация
- Поддержка путей алиасов
- Целевая версия ES2020

### Redux Store (`src/app/store/store.ts`)
- Настроен с `configureStore`
- Слайс для тарифов (`tariffsSlice`)

## 📝 Особенности реализации

1. **React Compiler**: Используется для автоматической оптимизации рендеринга.
2. **SVG Sprites**: Все иконки собраны в единый спрайт для уменьшения запросов.
3. **Кастомные хуки**: Логика вынесена в переиспользуемые хуки.
4. **Доступность**: Семантическая разметка и ARIA-атрибуты.
5. **Обработка ошибок**: Грациозная обработка ошибок загрузки данных.
